Creating a SharePoint List Column Using REST

April 20, 2018

Brent Ely

Send a REST call to any List or Library to create any type of column

Very useful!


sprLib.rest({
    url: '',
    type: 'POST'
    data: "{ __type: }"
})
.then(() => {
    console.log('Column created!');
})
